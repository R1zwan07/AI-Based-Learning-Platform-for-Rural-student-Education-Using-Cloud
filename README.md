# AI-Based Learning Platform for Rural Student Education using Cloud

## Overview
The AI-Based Learning Platform for Rural Student Education using Cloud is a web-based educational system developed to support students from rural areas by providing accessible digital learning resources.

The platform allows **administrators, faculty members, and students** to interact through a centralized system where educational materials such as videos, PDFs, quizzes, and AI chatbot assistance are available.

---

## Features

### Admin Module
- Admin authentication
- Add / update / delete faculty members
- View feedback from students
- Manage platform users

### Faculty Module
- Upload course materials
- Upload videos and PDF learning resources
- Add YouTube lecture links
- Manage and update uploaded materials
- Upload educational roadmaps for different classes

### Student Module
- Student registration and login
- Access course materials
- Download PDFs and videos
- View subject roadmaps
- Attempt quizzes
- Chat with AI chatbot for doubt solving
- Submit feedback

### AI Chatbot
- Integrated chatbot using Gemini API
- Multilingual support
- Maintains conversation history
